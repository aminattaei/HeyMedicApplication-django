## 1. Patient Dashboard

**URL:** `/dashboard/patient/`
**View:** `PatientDashboardView(LoginRequiredMixin, TemplateView)`

### Context data:
- `stats`: dict (total, pending, confirmed, completed, cancelled)
- `upcoming_appointments`: next 5 (pending/confirmed), with optional `?status=X` filter
- `recent_payments`: last 5 payments
- `recent_reviews`: last 5 reviews written by patient

### Template sections:
1. **Stats cards** - 4 cards: کل نوبت‌ها, در انتظار, تایید شده, انجام شده
2. **Upcoming appointments** - cards with doctor info, date/time, status badge, cancel button, filter by status
3. **Recent payments** - table: doctor, amount, status, date
4. **Recent reviews** - cards: doctor name, rating stars, comment

## 2. Doctor Dashboard (Full rewrite)

**URL:** `/dashboard/doctor/` (unchanged)
**View:** `DoctorDashboardView` - expanded context

### Additional context:
- `stats`: dict (total_appointments, today_count, total_patients, total_revenue)
- `total_revenue`: sum of successful payments for this doctor
- `recent_reviews`: last 5 reviews about this doctor
- `weekly_slots`: time slots for current week (Mon-Sun)
- `monthly_revenue`: last 6 months revenue data for chart
- `all_appointments` now supports `?status=X&search=Y` filters

### Template sections:
1. **Stats cards** - 4 cards: کل نوبت‌ها, نوبت امروز, کل بیماران, درآمد کل
2. **Revenue chart** - Chart.js line chart showing monthly revenue (last 6 months)
3. **Weekly calendar** - Grid view of current week's time slots (Mon-Sun columns)
4. **Time slot management** - Add form + list with delete button
5. **All appointments** - Table with status filter dropdown + patient name search + confirm/complete/cancel actions (JS)
6. **Recent reviews** - patient name, rating, comment
